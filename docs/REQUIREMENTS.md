
# Requirements

## Functional Requirements

- Detect API routes
- Detect database tables
- Detect service calls
- Generate AI summaries
- Store summaries in cache
- Render graph UI
- Provide API detail view
- Allow rescanning when code changes

## Non Functional Requirements

Performance
Graph must load instantly using cache.

Scalability
Support 100+ APIs.

Offline Usage
AI required only during scan phase.

Extensibility
Support multiple frameworks.

Security
Validate AI output before storing.
